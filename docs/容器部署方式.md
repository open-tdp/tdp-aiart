# 容器部署方式

## docker cli

```shell
docker run -d --name tdp-server \
    -p 7700:7700 \
    -v ./conf:/etc/tdp-aiart \
    -v ./data:/var/lib/tdp-aiart \
    rehiy/tdp-aiart
```

## docker-compose.yml

```ymal
version: "3.0"
services:
  tdp:
    image: rehiy/tdp-aiart:latest
    restart: always
    container_name: tdp
    volumes:
      - ./conf:/etc/tdp-aiart
      - ./data:/var/lib/tdp-aiart
    ports:
      - 7700:7700
```
